# Task 2: Real-Time Stock Price Tracker
## Objective
Simulate real-time stock price changes of a company TechCorp.
## Real-Life Use Case
This type of simulation is used in:
- Live trading dashboards
- Algorithmic trading systems
- Market data visualization
- Financial modeling and backtesting
## Requirements Implemented
- Initial stock price = 100
- Random change between –5 and +5 per update
- Total updates = 30
- Print each update:
- Timestamp
- Old Price → New Price
- Change (delta)
- Calculate at end:
- Highest price
- Total number of drops
- Longest continuous rising streak
